Creates small styled tags.

All colors with a light or light4 variant are used to generate tag styles.

Tags embedded in a link element get a hover color, which is the base or light2 variant respectively.

Selectors added:

.tag, .tag-[color]

Behaviour added:

a>.tag-[color]:hover, a.tag-[color]:hover
