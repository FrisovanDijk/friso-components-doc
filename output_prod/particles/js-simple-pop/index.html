Simple pop

A particle that appends the result of an API call to a parent container and hides it on click.

This is an alert appended to a fixed container:

JS function added:

simple_pop(container, data)

Requires:

static/components.js

Example code:

<button class="btn btn-primary w4" onclick='simple_pop("#alert-container", {
    "name": "alert",
    "vars": {
        "type": "success",
        "message": "Lorem ipsum"
    }
})'>Open success</button>

JS:

function simple_pop(container_id, data) {
    var child_id = Date.now();
    parseComponentHTMLChild(container_id, child_id, data);

    var pop = document.getElementById(child_id);

    pop.style.display = "block";
    pop.addEventListener("click", function(){
        pop.style.display = "none"
    });
}
