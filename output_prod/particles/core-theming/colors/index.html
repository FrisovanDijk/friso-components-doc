About these colors

Many web applications use standard patterns for color palettes. This implementation has been designed to allow for an extensible color palette. You can find the input values in _variables.scss.

Be aware that some components require specific modifiers such as light4 or light to work. Abiding to this pattern is a design choice I made for my components.

     bg         = background color
    -[color]
    -[color]-[tone]
    -wcag-[color]-[tone]

c          = color
    -[color]
    -[color]-[tone]

bg-wcag    = WCAG compliant contrast background
             and defined light and copy text colors.
    -[color]
    -[color]-[tone]

hgl        = Highlight (40% opacity of base)
    -[color]

hover-bg   = Hover background color
    -[color]
    -[color]-[tone]

hover      = Hover text color
    -[color]
    -[color]-[tone]

c-inherit  = inherit text color from parent

Default classes

Base colors don't require a [tone] modifier.
    bg-gray-base becomes bg-gray

    background color:       bg-[color]-[tone]
    text-color:             c-[color]-[tone]
    wcag text & background: bg-wcag-[color]-[tone]
    hover bg:               hover-bg-[color]-[tone]
    hover text:             hover-[color]-[tone]
    highlight bg:           hgl-[color]             only base colors

All available colors are listed below with high contrast text.
