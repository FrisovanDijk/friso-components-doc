Documentation guide

This documentation follows a specific ordering of elements, as shown below. Each element is included as needed where things are not self-explanatory.

So long as this pattern is followed, element naming won't matter.

Order

  • Rendered component
  • Modifier example
  • Behaviour example
  • Explaining text
  • Selectors added
  • Behaviour added
  • JS function(s) added
  • Required components
  • Example code
  • Source code
