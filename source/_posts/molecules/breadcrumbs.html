Breadcrumbs for accessible page navigation.

Example API call:

parseComponentHTML("#breadcrumbs", {
    "name": "breadcrumbs",
    "vars": {
        "items": [
            {"name": "molecule", "url": "{{site.url}}/blog/tags/molecule"},
            {"name": "{{page.title}}", "current?": true}
        ]
    }
});
